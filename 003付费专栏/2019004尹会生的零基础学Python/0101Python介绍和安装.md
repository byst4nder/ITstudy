## 01. Python 的发展历史和版本

1990 年：Python 诞生

2000 年：Python2.0 发布
 
2008 年：Python3.0 发布

2010 年：Python2.7 发布（最后一个 2. X 版本）

Python 有官方版和流行版之说，官方版只有最基础的东西，比如解释器、基础库；Python 最流行的发行版 Anaconda。
